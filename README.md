# Eventify 🎉

Eventify is a MERN Stack Event Management System where users can register, login, and manage events.

## Features

- User Registration
- User Login Authentication using JWT
- Create Events
- View Events
- Update Events
- Delete Events
- MongoDB Database Integration
- REST API Backend

---

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

### Tools
- Postman
- Git & GitHub
